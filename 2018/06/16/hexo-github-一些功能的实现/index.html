来必力

来必力简单，没有广告，支持多种账号登录

1.登陆注册来必力
2.在右上角可以进入管理界面，可以进行各种配置

4.在theme/主题名/_config.yml中添加配置信息

5.在themes/xxxx/layout/_partial/article.ejs中，配置评论
